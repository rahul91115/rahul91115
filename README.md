## 🚀 Featured Projects

### 🔌 [canfd-eth-gateway](https://github.com/rahul91115/canfd-eth-gateway)
Lightweight C++ gateway bridging **CAN-FD ↔ Ethernet** for automotive diagnostics and telematics.
- C++17, no external dependencies · Linux SocketCAN · Binary serialization (~80 bytes/packet)
- Real-time scheduling (`SCHED_FIFO`) · systemd service · Yocto-compatible

### 🧬 [digital-twin-monitoring](https://github.com/rahul91115/digital-twin-monitoring)
ROS2-based **digital twin** monitoring system for real-time state synchronization of physical assets.
- C++ / Python · ROS2 architecture · Sensor fusion pipeline

### 🎯 [object-detection-and-depth-estimation](https://github.com/rahul91115/object-detection-and-depth-estimation)
Fused **YOLOv3 + LiDAR** pipeline for simultaneous object detection and depth estimation.

### 📡 [radar_interference_classifier](https://github.com/rahul91115/radar_interference_classifier)
ML-based classifier for **radar interference patterns** — signal processing meets machine learning.

### 🔄 [MPC_Inverted_Pendulum](https://github.com/rahul91115/MPC_Inverted_Pendulum)
Model Predictive Control implementation for the classic **inverted pendulum** stabilization problem.
